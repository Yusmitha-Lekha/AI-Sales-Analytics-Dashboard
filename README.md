# AI-Enhanced Sales Analytics Dashboard (SQL + Python + Power BI)

## Overview
This project analyzes a retail/superstore sales dataset end-to-end using:
- **SQL** for business queries and KPI calculations
- **Python** for data cleaning, EDA, and forecasting
- **Power BI** for interactive dashboards
- **AI (optional)** for automatic executive summaries

The goal is to demonstrate a modern **AI-enabled Data Analyst** workflow: from raw data → insights → dashboard → executive narrative.
